## Project Overview

**mg-detect** implements unsupervised breast lesion detection via bilateral asymmetry analysis. The system learns what normal bilateral differences look like (left vs. right breast), then flags unusual differences as anomalies at test time — without requiring lesion annotations during training.

## Architecture

The system has four sequential components, described in detail in `plan/CycleCUT_Complete_Guide.md`:

1. **CycleCUT Generator** — learns image-to-image translation between left and right breast (CycleGAN + contrastive/CUT loss). Trains on normal pairs only.
2. **Residual Extraction** — computes `r_A = |A - G_BA(B)|` and `r_B = |B - G_AB(A)|` per patient.
3. **Patch Encoder** — CNN φ that maps 32×32 residual patches → 128-dim feature vectors; trained on normal residuals.
4. **Normalizing Flow** — learns the distribution of normal residual features; anomaly score = `-log p_flow(φ(patch))`.

At test time the output is a per-patch heatmap: high score → possible lesion.

Two experimental plans exist:
- **Plan A** — Fully unsupervised, train on normal pairs only. Implement this first.
- **Plan B** — Extends Plan A with synthetic lesion corruptions for semi-supervised training.

## Data

- **VinDr-Mammo** (primary): `data/vindr/` CSVs + raw DICOM/PNG images under `data/vindr-full/`
  - `finding_annotations.csv` — per-finding annotations with bounding boxes
  - `breast-level_annotations.csv` — breast-level BI-RADS density
  - `no_findings.csv` — pre-filtered list of 18,232 normal cases (generated by `notebooks/vindr_ds.ipynb`)
- Uses **CC view only** (craniocaudal) — most symmetric, avoids pectoral muscle present in MLO.
- Exclusions: implants, post-surgical, bilateral cancer, poor-quality images.
- Split: 70% normal for training, 15% normal for validation, 15% normal + all unilateral lesion cases for test.

## Key Implementation Notes

- Images are standardized to 512×512×1 (grayscale).
- Breast region must be cropped/registered before training — see `notebooks/registration.ipynb` for the OpenCV-based approach.
- The normalizing flow scores normal vs. anomalous patches; it never sees lesion labels during training.
- Bilateral cancer cases (lesion on both sides) must be excluded from test positives as they break the symmetry assumption.
